# CJXplorer

Агентский сервис для исследования и оценки клиентских путей (Customer Journey) по критериальной модели.

## Пользовательские сценарии

1. **Оценка по скриншотам** — пользователь загружает набор скриншотов CJ, агент оценивает путь по критериальной модели
2. **Автономное исследование** — пользователь указывает CJ и целевые приложения/сайты, агент самостоятельно проходит путь и оценивает его
3. **Оценка макетов** — пользователь загружает макет из Pixso/Figma или даёт ссылку, агент оценивает CJ по критериальной модели

## Структура проекта

```
docs/
├── architecture/       # Архитектурные документы, диаграммы, ADR
├── user-stories/       # Детализация пользовательских сценариев
└── knowledge-base/     # Критериальная модель, эвристики оценки
```
